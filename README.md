# Hermes Agent Docker Installer\n\n[![CI](https://github.com/lunaticbugbear/hermes-docker-installer/actions/workflows/ci.yml/badge.svg)](https://github.com/lunaticbugbear/hermes-docker-installer/actions/workflows/ci.yml)\n[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)\n\nPublic-quality cross-platform installer for running **Hermes Agent** inside Docker.\nOne command to install, update, and manage -- no manual setup, no dependency conflicts, no environment pollution.\n\n---\n\n## Features\n\n| Feature | Description |\n|---------|-------------|\n| Docker-isolated runtime | Hermes runs in a container -- zero host pollution |\n| Persistent data | Config, sessions, memory survive restarts via Docker volume |\n| API Server | OpenAI-compatible gateway on localhost:8642 with auto-generated Bearer key |\n| Cross-platform | Linux, macOS, Windows (WSL/Git Bash), Windows native (PowerShell) |\n| Browser tools | Optional Playwright + Chromium for web automation skills |\n| Helper CLI | bin/hermes-docker for start/stop/logs/cli/shell/update/reset |\n| Clean uninstall | One command -- with optional data volume cleanup |\n| CI-tested | ShellCheck, syntax validation, Docker compose config, and smoke test |\n\n## Architecture\n\n```\nHost Machine\n\n~/.hermes-docker/\n  .env                  Provider keys + config\n  Dockerfile            Multi-stage build\n  docker-compose.yml    Stack definition\n  bootstrap.sh          Container entrypoint\n  healthcheck.sh        Docker HEALTHCHECK\n  bin/hermes-docker     CLI helper\n  workspace/            Bind-mounted to /workspace\n\nDocker container:\n  Hermes Agent CLI + Skills\n  API Server (port 8642)\n  /workspace (bind mount)\n  /root/.hermes (Docker volume)\n```\n\n## Requirements\n\n### Linux / WSL\n```bash\ncurl -fsSL https://get.docker.com | sh\nsudo usermod -aG docker $USER\n# Log out and back in\n```\n\n### macOS\nInstall Docker Desktop.\n\n### Windows\nInstall Docker Desktop with WSL2 backend.\n\n## Quick Install\n\n### Linux / macOS / WSL\n```bash\ncurl -fsSL https://raw.githubusercontent.com/lunaticbugbear/hermes-docker-installer/main/install.sh | bash\n```\n\n### Windows PowerShell\n```powershell\nirm https://raw.githubusercontent.com/lunaticbugbear/hermes-docker-installer/main/install.ps1 | iex\n```\n\n## Usage\n\n```bash\ncd ~/.hermes-docker\n./bin/hermes-docker start   # Start container\n./bin/hermes-docker cli     # Open Hermes CLI\n./bin/hermes-docker logs    # Follow logs\n./bin/hermes-docker url     # Print API URL\n./bin/hermes-docker key     # Print API key\n./bin/hermes-docker update  # Rebuild + restart\n./bin/hermes-docker reset   # Remove container + data\n./bin/hermes-docker shell   # Open container shell\n```\n\n## Options\n\n| Option | Default | Description |\n|--------|---------|-------------|\n| --dir PATH | ~/.hermes-docker | Install directory |\n| --provider PROVIDER | openrouter | Model provider |\n| --model MODEL | deepseek/deepseek-v4-flash:free | Model name |\n| --port PORT | 8642 | API server port |\n| --name NAME | hermes-agent | Docker Compose project prefix |\n| --no-start | -- | Build image but don't start container |\n| --no-browser | -- | Skip Playwright/Chromium |\n| --skip-build | -- | Generate files only (CI mode) |\n| --force | -- | Overwrite existing files |\n| --uninstall | -- | Stop and remove stack |\n\n## Non-Interactive Install\n\n```bash\nHERMES_NONINTERACTIVE=1 OPENROUTER_API_KEY=*** bash install.sh --provider openrouter --model deepseek/deepseek-v4-flash:free\n```\n\n### Provider examples\n\n```bash\nHERMES_NONINTERACTIVE=1 ANTHROPIC_API_KEY=*** bash install.sh --provider anthropic --model claude-sonnet-4\nHERMES_NONINTERACTIVE=1 OPENAI_API_KEY=*** bash install.sh --provider openai --model gpt-4.1\nHERMES_NONINTERACTIVE=1 GOOGLE_API_KEY=*** bash install.sh --provider google --model gemini-2.0-flash\nHERMES_NONINTERACTIVE=1 DEEPSEEK_API_KEY=*** bash install.sh --provider deepseek --model deepseek-chat\n```\n\n## Config Reference\n\n### .env variables\n\n| Variable | Purpose |\n|----------|---------|\n| MODEL_PROVIDER | Provider name |\n| MODEL_NAME | Model identifier |\n| API_SERVER_PORT | API gateway port |\n| API_SERVER_KEY | Auto-generated Bearer token (48 hex chars) |\n| INSTALL_BROWSER | 1 = with Playwright + Chromium |\n| OPENROUTER_API_KEY | Key for OpenRouter |\n| ANTHROPIC_API_KEY | Key for Anthropic |\n| OPENAI_API_KEY | Key for OpenAI |\n| GOOGLE_API_KEY | Key for Google Gemini |\n| GEMINI_API_KEY | Alias for Google Gemini |\n| DEEPSEEK_API_KEY | Key for DeepSeek |\n| CUSTOM_API_KEY | Key for custom endpoint |\n| CUSTOM_BASE_URL | Base URL for custom endpoint |\n\n## Uninstall\n\n```bash\n./uninstall.sh                 # Stop, keep data\nREMOVE_DATA=1 ./uninstall.sh    # Also remove volume\nREMOVE_DATA=1 REMOVE_FILES=1 ./uninstall.sh  # Remove everything\n```\n\n## Docker Image\n\nMulti-stage build:\n- **builder**: python:3.12-slim-bookworm + build tools (git, build-essential)\n- **runtime**: python:3.12-slim-bookworm + runtime packages only\n\nImage sizes (approximate):\n| Configuration | Size |\n|---------------|------|\n| Without browser | ~390 MB |\n| With browser | ~840 MB |\n\n## Troubleshooting\n\n| Symptom | Fix |\n|---------|-----|\n| docker: not found | Install Docker Desktop / Engine |\n| Docker not reachable | Start Docker (sudo systemctl start docker) |\n| Port 8642 in use | Run with --port 8643 |\n| Model calls fail | Check API key in .env |\n| Container crashes | Check .env provider config |\n\n## Security\n\n- API keys in .env -- do not commit to Git\n- API server key auto-generated on install\n- Port mapped to localhost by default\n- For external access, use reverse proxy with TLS
